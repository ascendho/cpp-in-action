# cpp-in-action

| 书籍章节                | 文件夹名                          | 设计说明                                                                 |
|-------------------------|-----------------------------------|--------------------------------------------------------------------------|
| 绪论                    | `introduction`                    | 简洁明了，直接对应“绪论”主题，无冗余信息。                               |
| 第1章 C和C++基础        | `c_cpp_basics`                    | 用“c_cpp”明确语言范围，“basics”对应“基础”，精准覆盖章节核心知识点。       |
| 第2章 对象生存期和RAII   | `object_lifetime_raii`            | 提取“对象生存期（object_lifetime）”和“RAII”核心术语，RAII为C++高频缩写，无需展开。 |
| 第3章 值类别和移动语义   | `value_categories_move_semantics`  | 完整对应“值类别”和“移动语义”两大核心主题，术语准确，无歧义。             |
| 第4章 模板和自动类型推导 | `templates_auto_type_deduction`   | 突出“模板（templates）”和“自动类型推导（auto_type_deduction）”，覆盖章节核心。 |
| 第5章 字符串            | `strings`                         | 章节主题单一明确，用复数形式简洁表示，符合STL中`string`的常用表述习惯。   |
| 第6章 函数对象          | `function_objects`                | 直接对应章节标题，包含lambda、function等核心内容，名称全覆盖。            |
| 第7章 标准容器          | `standard_containers`              | 明确“标准容器（standard_containers）”，区分自定义容器，精准对应STL容器主题。 |
| 第8章 迭代器            | `iterators`                       | 章节主题单一，用复数形式简洁表示，涵盖各类迭代器（输入、双向、随机访问等）。 |
| 第9章 标准算法          | `standard_algorithms`              | 对应“标准算法”，包含并行算法、范围算法等，名称统一且精准。               |
| 第10章 视图             | `views`                           | 简洁对应“视图”主题，涵盖`string_view`、`span`、C++20范围视图等内容。      |
| 第11章 智能指针         | `smart_pointers`                  | 直接对应章节标题，包含`unique_ptr`、`shared_ptr`、`weak_ptr`等核心内容。  |
| 第 12 章 现代 C++ 的一些重要改进      | `modern_cpp_important_improvements` | 提取 “现代 C++（modern_cpp）” 和 “重要改进（important_improvements）” 核心关键词，完整覆盖章节主题（如 constexpr、tuple、chrono 等）。 |
| 第 13 章 契约和异常                   | `contracts_and_exceptions`          | 直接对应章节核心内容 “契约（contracts）” 和 “异常（exceptions）”，简洁无冗余，术语准确。 |
| 第 14 章 optional、variant 和错误处理 | `optional_variant_error_handling`   | 突出章节核心组件 “optional”“variant”，搭配 “错误处理（error_handling）”，精准覆盖非异常错误处理的核心内容。 |
| 第 15 章 传递对象的方式               | `object_passing_methods`            | 核心关键词 “对象传递（object_passing）”+“方式（methods）”，贴合章节对传参优化（值传参、移动优化等）的讨论。 |
| 第 16 章 并发编程                     | `concurrency_programming`           | 直接对应章节主题 “并发编程”，涵盖线程、锁、原子量、内存序等核心内容，术语通用且专业。 |
| 第 17 章 异步编程                     | `asynchronous_programming`          | 区分 “并发” 与 “异步”，用 “asynchronous_programming” 精准对应章节的协程、Asio、future 等内容。 |
| 第 18 章 探索 C++ 的工具              | `exploring_cpp_tools`               | “探索（exploring）” 对应章节标题，“C++ 工具（cpp_tools）” 涵盖编译器、Clang 系列工具、Sanitizer 等核心内容，贴合主题。 |
